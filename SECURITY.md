# Security Policy

## Supported Versions

Only the latest minor release receives security updates.

| Version | Supported |
| ------- | --------- |
| 1.x.x   | ✅       |
| < 0.4   | ❌        |

## Reporting a Vulnerability

**Do not open a public issue for security vulnerabilities.**

Email **fearlesslymediagroup@gmail.com** with:

- A description of the vulnerability
- Steps to reproduce
- Affected version(s)
- Your assessment of the impact

You should get an initial response within 24 hours. If confirmed, a fix will land in the next patch release and you will be credited in the release notes (unless you prefer to stay anonymous).
