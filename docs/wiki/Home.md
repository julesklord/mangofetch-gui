# 🥭 MangoFetch Wiki

Welcome to the official documentation for **MangoFetch**, the high-performance, universal terminal download manager.

MangoFetch is engineered for users who demand speed, flexibility, and a beautiful terminal experience. Whether you're a CLI power user or prefer an interactive TUI, MangoFetch provides a robust engine to download media from over 1000+ platforms, including YouTube, Instagram, TikTok, and many more, as well as supporting Torrents and P2P transfers.
